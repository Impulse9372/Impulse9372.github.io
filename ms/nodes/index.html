Node Combinations

For example, 3 for Night Walkers, 4 for Hayatos, and 6 for Kannas.
List nodes that are always required and need to be boosted twice.
For Adeles that have 6 required skills, you'd put "A, B, C, D, E, F".
For Hayatos that have 5 required skills and a choice for their last one, you'd put "A, B, C, D, E"; F = Sudden Strike; G = Summer Rain.
For Night Walkers that have 4 required skills and a choice for their last one, you'd put "A, B, C, D"; E = Dominion; F = Shadow Stitch.
Uncheck for classes like Night Walker which only needs to boost their last skill once.
